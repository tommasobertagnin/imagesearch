Image Search

This is a basic image search abstraction layer.

Example usage

  1. search: https://imagesearchal.herokuapp.com/api/search/sea
  2. latest: https://imagesearchal.herokuapp.com/api/latest

User Stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string (api endpoint http://imagesearchal.heroku.com/api/search/...)
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.
